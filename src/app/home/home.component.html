Guess Personal Name, Origin or Ethnicity

Our API will guess the personal name's country of origin, to prioritize which international phone number formats to try. For example, Andrea Rossini is an Italian name, so why not try an Italian phone number first? Our machine learning model (ML) combines both the personal name features, as well as the phone number formatting in order to validate to a E164 number with ~95% precision overall.
